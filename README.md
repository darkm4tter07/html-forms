This is task of UnifyApps Frontend Onboarding

Task Includes

Create a HTML page with a form element.

Add the following fields:

1. Name
2. Age (Should only accept number)
3. Gender (Should have Male, Female, Other and either can be selected once)
4. Country (A dropdown with these countries: India, USA, Japan, China)(Can select one)
5. Technological stack (Add these fields: HTML, CSS, JS, React, React Native) (Can select multiple)
6. Address (The user should be able to add multiple lines of address)
7. Telephone (Should only accept telephone numbers)
8. Email (Should only accept email id)

Add two buttons below the Form
1. Reset (On Clicking this, the form values must reset to initial state)
2. Submit (On Click must perform the below actions)
- Validate that all the fields are filled
- Validate that all fields have respective values of their types

After successful validation print the form data below the form in the tabular form.
